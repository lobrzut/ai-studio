## About section (long)

**AI Studio** is a monorepo for local creative AI on **Windows** (portable folder) and **Linux** (Debian server).

It bundles:

- **ComfyUI** — image/video workflows and ComfyUI-Manager
- **ACE-Step** — music generation (Gradio)
- **Shared dashboard** (`:7880`) — PL/EN UI, service control, GPU helpers, Comfy output gallery, drag-and-drop audio post-production

**Windows:** `Install.bat` then `Start.bat` — no system Python required; copy the folder to USB or another PC.

**Linux:** `curl -fsSL https://raw.githubusercontent.com/lobrzut/ai-studio/main/linux/bootstrap.sh | sudo bash`
