## About section (long)

AI Studio Portable (**Local edition**) is a self-contained Windows launcher and dashboard for creative AI workflows on one machine.

It bundles:

- **ComfyUI** for image/video workflows
- **ACE-Step** for music generation
- **Toolkit dashboard** (`:7880`) with service control, GPU helpers, Comfy output gallery, and drag-and-drop audio post-production

Designed for copy-to-USB portability: run `Install.bat` once, then `Start.bat` on any compatible machine. No system-wide Python install required.
